# Descrição

Este projeto utiliza uma **função simples para identificar números pares** dentro de uma lista.
É um exemplo prático de uso de funções, estruturas de repetição e operadores matemáticos em Python.

## Funcionalidades

1. **Verificação de números pares** – a função `is_even()` retorna `True` se o número for par.
2. **Iteração sobre lista** – percorre uma lista de números de 1 a 10 e exibe apenas os pares.

## Objetivo do Projeto

Praticar a criação de funções e o uso de **condições e loops em Python**, reforçando a lógica para filtragem de dados em listas.
